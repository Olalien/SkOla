⚙️ Innstillinger

🔑 Anthropic API-nøkkel
Nødvendig for AI-generering. Lagres kun lokalt i nettleseren.
Hent gratis nøkkel på console.anthropic.com → API Keys → Create Key
🔐 Endre lærerpassord
Passordendringen gjelder kun denne nettlesersesjonen.
🖥️ Microsoft-integrasjon
Koble OlaSkole til Microsoft 365 så elever kan eksportere arbeid direkte til OneNote og Word Online med sin skolekonto.
🔑 Microsoft App-ID (Azure)
📋 Slik setter du opp: Azure Portal → App registrations → New registration → Legg til http://localhost som Redirect URI (SPA). Aktiver Files.ReadWrite og Notes.ReadWrite i API permissions.
✅ Koblet til Microsoft 365 som –
Når tilkoblet kan elever sende arbeid til OneNote Klasse-notatblokk og OneDrive/Word Online direkte fra arbeidsøkten.
🗄️ Database – Supabase
⏳ Sjekker…
⚠️ Databasetabellene er ikke opprettet!
Gå til supabase.com → ditt prosjekt → SQL Editor, kopier SQL-skjemaet under og kjør det. Gjør dette én gang.
⚙️ Avansert – endre tilkobling
⚠️ Ingen database koblet til! Live Quiz virker kun på samme nettleser uten Supabase.
Koble til Supabase under ⚙️ Innstillinger → Database.
